# Eismark Campaign Documents

This repository contains the Eismark Canon Document archive, readable guide, audit/index files, and the companion website source for browsing the campaign archive.

## Key Folders

- `outputs/` - canonical ECD archive, readable guide, audit register, protocol addendum, and master index.
- `website/` - local website/source files for browsing the campaign material.
- `website/content/archive/` - website-facing copies of the archive documents.

Generated dependencies and build artifacts are intentionally excluded from version control.
